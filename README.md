# ordenamiento_ecologico

## Descripción
Conjunto de scripts en R para el procesamiento vectorial y raster, integrados con software de Sistemas de Información Geográfica (SIG), como QGIS. Este repositorio facilita la elaboración de cartografía útil para la creación de mapas de riesgos y ordenamientos ecológicos.

## Funcionalidades principales
- Procesamiento y análisis de datos vectoriales y raster.
- Integración con QGIS para mejorar la eficiencia en el manejo de datos geoespaciales.
- Generación de mapas para evaluar riesgos ambientales y planificar ordenamientos ecológicos.

## Uso
Los scripts están diseñados para ser utilizados junto con software SIG de código abierto, principalmente QGIS, para:
- Procesar datos espaciales complejos de manera integrada.
- Crear mapas temáticos que sirvan como herramientas de apoyo en la gestión ecológica y ambiental.

## Requisitos
- R (lenguaje de programación utilizado para los scripts).
- QGIS u otro software SIG compatible para la visualización y manejo de los datos generados.

## Instalación
Clonar el repositorio:
```
git clone https://github.com/OlivaresLD/ordenamiento_ecologico.git
```
